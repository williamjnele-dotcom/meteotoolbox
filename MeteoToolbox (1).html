ABOUT METEOTOOLBOX

MeteoToolbox is a free, open severe weather data tool built for storm spotters, chasers, and weather enthusiasts. It pulls real-time data from NOAA's Storm Prediction Center, IEM NEXRAD, Open-Meteo, and the National Weather Service API — no account, no API keys, no cost.

What it does:
Every tab serves a specific purpose in your severe weather workflow. The SPC Outlooks tab shows live convective outlooks for Days 1–8 with tornado, wind, and hail probability polygons pulled directly from NOAA's ArcGIS servers. The NWS Radar tab gives you real-time NEXRAD composite radar with base reflectivity, echo tops, velocity, and 1-hour precip — plus a live loop and severe weather warning overlays. The Region Map lets you click any major storm corridor to instantly load model parameters for that area. The Model Data tab fetches live GFS, HRRR, NAM, or NBM forecast data for any US city, coordinates, or state and displays CAPE, Lifted Index, SRH, EHI, SCP, STP, bulk shear, and lapse rates with color-coded threat levels. The Active Alerts tab maps all live NWS watches and warnings with live polygon overlays and a warning counter. The States & Sub-Regions tab is a clickable reference map for every data collection area. The Model Runs tab shows live countdowns to the next 0Z, 6Z, 12Z, and 18Z model initializations so you always know when fresh data will be ready.

How to use it:
Start with SPC Outlooks to check if severe weather is in the forecast. If there's a risk area, open Model Data and load your target city or state to review the thermodynamic and kinematic parameters. Use the Region Map for a quick one-click load of a storm corridor. Once storms are ongoing, switch to Radar and enable the warnings overlay to track active cells and active warnings simultaneously. Keep Active Alerts open in the background — it auto-refreshes every 2 minutes and can send browser notifications for new tornado warnings, severe thunderstorm warnings, and SPC Mesoscale Discussions.

Data sources:
SPC convective outlooks · IEM NEXRAD WMS radar tiles · Open-Meteo GFS/HRRR/NAM/NBM API · NWS alerts API (api.weather.gov) · NWS products API (KWNS MCD) · IEM state border tiles

⚠ MeteoToolbox is for situational awareness and educational use only. For life-safety decisions always rely on official NWS warnings at weather.gov. Never use this tool as a substitute for official emergency alerts.
